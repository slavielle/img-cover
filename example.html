Image-cover in action ...

resize window to see the cover in action ...

 
Cover using object-fit CSS property ... Great but not supported by IE11
Cover using js and background ... Great workaround for the same result fully compatible with IE11
Cover using a defined width and height values 
img-cover.scss +
custom scss :
    .img-cover-1-b {
      width: 100%;
      height: 300px;
    }
html:
    <div class="img-cover-1-b img-cover">
      <img src="images/img_01.jpg">
    </div>
Cover-image using defined width and keeping a 3:4 with/height ratio 
img-cover.scss +
custom scss :
    .img-cover-2-b {
      @include img-cover-fit-ratio(100%, 3/4);
    }
html:
    <div class="img-cover-2-b img-cover">
      <img src="images/img_01.jpg">
    </div>

Advanced stuff

Image-cover workaround allows to be easily impoved with some cool stuff

lazy loading

Cover-image lazy loading using verlok / lazyload (when you scroll down the image after page loaded, the picture is lazy loaded and appears with a fade in effect) 

html:
<div class="img-cover-fx-example img-cover img-cover--fx-load-fadein">
    <div class="img-cover__inner">
        <img data-src="images/img_01.jpg">
    </div>
</div>
scss:
  img-cover.scss + custom scss:
    .img-cover-fx-example {
      @include geodis-fit-ratio-image(20rem, 3/4);
    }
    .img-cover {
      .img-cover__inner {
        transition: opacity 1s ease-in-out;
      }
      &.img-cover--fx-load-fadein .img-cover__inner {
        opacity: 0;
      }
      &.img-cover--fx-load-fadein .img-cover__inner[style] {
        opacity: 1;
      }
    }

Hover effect

Cover-image with a zoom effect on hover

html:
<div class="img-cover-fx-example img-cover img-cover--fx-hover-zoom">
    <div class="img-cover__inner">
        <img data-src="images/img_01.jpg">
    </div>
</div>
scss:
  img-cover.scss + custom scss:
    .img-cover-fx-example {
      @include geodis-fit-ratio-image(20rem, 3/4);
    }
    .img-cover {
      .img-cover__inner {
        margin .3s cubic-bezier(0, 1.9, .75, .18),
        width .3s cubic-bezier(0, 1.9, .75, .18),
        height .3s cubic-bezier(0, 1.9, .75, .18);
      }
      &:hover.img-cover--fx-hover-zoom .img-cover__inner {
        width: calc(100% + 2rem);
        height: calc(100% + 2rem);
        margin-left: -1rem;
        margin-top: -1rem;
      }
    }
